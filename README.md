# HLS-Calculator
This calculator can receive two signed five with binary inputs and can perform eight operations on them, including addition, subtraction, logical bitwise AND, logical bitwise OR, logical bitwise NAND, logical bitwise XOR, arithmetic negation and logical bitwise inverse. Then It shows the results on Seven Segments using decimal and HEX format
